## When Triggered

Run this on heartbeat events or when Josh asks for a status update.

## Checks

### Inbox

- Urgent or time-sensitive emails needing a reply
- Actionable unread messages
- Skip newsletters and noise

### Calendar

- Today's upcoming events
- Scheduling conflicts
- Anything in the next 24h that needs prep

### Tasks

- Overdue or due-today items
- Anything stuck for 3+ days
- Note blockers if known

### Health & Habits *(morning only)*

- Daily habit or medication reminders
- Medical appointments within 48h

### Finance

- Bills or payments due within 3 days

## Memory Maintenance

Use heartbeats to keep memory clean:

1. Review recent daily notes
2. Promote significant learnings to `MEMORY.md` (merge in-place, don't append)
3. Remove stale entries from `MEMORY.md`
4. If nothing worth promoting — skip it

## Rules

- If nothing needs attention: **HEARTBEAT_OK**
- Don't repeat alerts from a prior heartbeat unless status changed
- Don't infer tasks from old conversations — only act on current data
- Don't editorialize — surface facts, not opinions
- Keep the whole heartbeat under 15 lines when possible
